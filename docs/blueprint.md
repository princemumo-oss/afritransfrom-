# **App Name**: LinkUp

## Core Features:

- User Profiles: Create and manage personal profiles with information and customizable details.
- Main Feed: View posts from friends in a real-time, updating feed.
- Post Creation: Create new text or photo posts to share with friends. LLM based tool to filter out disallowed content and recommend hashtags for increasing discoverability.
- Friend Connections: Send, accept, and manage friend requests to build a social network.
- Direct Messaging: Send private messages to friends in real time.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) for a calm, trustworthy feel.
- Background color: Light gray (#EEEEEE) for a clean and modern look.
- Accent color: Slightly darker blue (#42A5F5) to highlight interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a clean and modern look.
- Use simple, outlined icons for easy recognition and a modern aesthetic.
- Employ a clean, card-based layout for posts and profiles, enhancing readability.
- Incorporate subtle animations for likes, comments, and new messages to improve user engagement.
